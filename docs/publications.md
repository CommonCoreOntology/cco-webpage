<h1>Publications</h1>

<p>
  This page brings together key publications that explain what the Common Core Ontologies are, why they were developed, and how they are used in practice. The list is intentionally curated to focus on work that directly presents or applies CCO.
</p>

<h2>CCO White Papers</h2>

<p>
  <strong>
    <a href="https://www.nist.gov/system/files/documents/2021/10/14/nist-ai-rfi-cubrc_inc_004.pdf" class="custom-color">
      Overview of Common Core Ontologies
    </a>
  </strong> (2021)<br/>
  <em>CUBRC, Inc.</em><br/>
  Provides a high-level overview of the Common Core Ontologies, including their modular design, scope, and role as a mid-level ontology supporting interoperability across domains.
</p>

<p>
  <strong>
    <a href="https://www.nist.gov/system/files/documents/2021/10/14/nist-ai-rfi-cubrc_inc_003.pdf" class="custom-color">
      Modeling Information with Common Core Ontologies
    </a>
  </strong> (2021)<br/>
  <em>CUBRC, Inc.</em><br/>
  Describes how the Common Core Ontologies model information, including distinctions between information content entities, their bearers, and representational patterns.
</p>

<p>
  <strong>
    <a href="https://philarchive.org/archive/COXTSD-2" class="custom-color">
      An Overview of the Common Core Space Domain Ontologies
    </a>
  </strong> (2019)<br/>
  <em>Alexander P. Cox et al.</em><br/>
  Introduces a family of space-domain ontologies that extend the Common Core Ontologies to represent spacecraft, space objects, and space-related events.
</p>

<h2>Journal and Conference Publications</h2>

<p>
  <strong>
    <a href="https://www.utwente.nl/en/eemcs/fois2024/resources/papers/jensen-et-al-the-common-core-ontologies.pdf" class="custom-color">
      The Common Core Ontologies
    </a>
  </strong> (2024)<br/>
  <em>J. Beverley, M. Jensen, G. De Colle, S. Kindya, C. More, A. P. Cox</em><br/>
  Presents the architecture, design principles, and intended uses of the Common Core Ontologies as a mid-level ontology suite.
</p>

<p>
  <strong>
    <a href="https://arxiv.org/abs/2404.17757" class="custom-color">
      Middle Architecture Criteria
    </a>
  </strong> (2024)<br/>
  <em>J. Beverley, G. De Colle, M. Jensen, C. Benson, B. Smith</em><br/>
  Proposes criteria for evaluating mid-level ontologies and uses the Common Core Ontologies as a motivating and evaluative example.
</p>

<p>
  <strong>
    <a href="https://philpapers.org/archive/MORJDO.pdf" class="custom-color">
      Joint Doctrine Ontology
    </a>
  </strong> (2020)<br/>
  <em>John M. Beverley et al.</em><br/>
  Describes a domain ontology aligned with the Common Core Ontologies for representing joint military doctrine and operational concepts.
</p>

<p>
  <strong>
    <a href="https://www.spiedigitallibrary.org/conference-proceedings-of-spie/10635/2307719/A-common-core-based-cyber-ontology-in-support-of-cross/10.1117/12.2307719.short" class="custom-color">

    </a>
  </strong> (2018)<br/>

  Demonstrates how the Common Core Ontologies can be extended to support cyber situational awareness across heterogeneous systems.
</p>
